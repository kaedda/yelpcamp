# yelpcamp

PROJECT TITLE: YELPCLONE FULL-STACK DISCOVERY PLATFORM

OVERVIEW This is a full-stack web application designed to help users discover, rate, and review local businesses. This project was built as a comprehensive learning exercise to master the MERN stack, RESTful API design, and user authentication. The goal was to move beyond basic applications and build a complex system featuring a responsive React frontend, a secure Node and Express backend, and a flexible MongoDB database.
